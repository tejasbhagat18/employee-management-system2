About Our Project

Employee Management System

Employee Management System is a full-stack web application designed to simplify employee record management within an organization.

This project provides secure authentication, role-based access, employee profile management, and admin tools for managing employee details.

Project Objectives

  • Manage employee records efficiently.
  • Provide separate Admin and Employee dashboards.
  • Implement secure login and authentication.
  • Perform CRUD operations on employee data.
  • Create responsive and user-friendly interfaces.
  • Maintain data using SQLite database.

Admin

Manage employees, departments, salaries, and employee accounts.

Employee

View personal profile and update limited information securely.

Security

Password hashing, session management, and role-based access.
